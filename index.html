My project there involved the creation of a basic interactive chatbot for College Admissions

  • The chatbot system utilizes a pre-trained neural network model loaded from a file ('chatbotmodel.h5') using TensorFlow/Keras.
  • Upon receiving a user message, the system preprocesses it and generates a bag-of-words representation using the bag_of_words function.
  • Using the loaded model, the system predicts the intent of the message by calculating the probabilities of each intent class. If a threshold is surpassed, the intent with the highest probability is selected. Finally, a random response associated with the predicted intent is retrieved from the intents defined in the intents.json file.
